# HV-devops-environment-setup

# System Monitoring, User Management & Backup Automation

## Overview
This repository contains all deliverables for the DevOps assignment including:

- System monitoring setup
- User management & access control
- Automatic backup scripts for Apache & Nginx
- Verification logs
- Report & screenshots

---

## Repository Contents

### 1. `/report`
Contains the full assignment report (`PDF`).

### 2. `/scripts`
Automated backup scripts:
- `apache_backup.sh`
- `nginx_backup.sh`

### 3. `/screenshots`
Evidence of task completion:
- Monitoring outputs
- User creation
- Permissions validation
- Backup job verification

### 4. `/logs`
Includes sample verification logs generated after backups.
